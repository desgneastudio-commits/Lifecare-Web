# Lifecare-Web

A modern, high-end healthcare website experience built with React, Vite, Tailwind CSS, and immersive 3D visuals. This project is designed to reflect a premium medical brand with a calm, futuristic, and trustworthy user experience.

This project was created as part of a portfolio-focused showcase for Desgnea, highlighting strong UI/UX design, motion-driven interfaces, and modern frontend development practices.

## ✨ Highlights

- Elegant landing experience with cinematic hero visuals
- Interactive department showcase and medical service pages
- Doctor profiles and patient dashboard UI
- Immersive virtual hospital tour powered by 3D scenes
- Smooth animations, glassmorphism, and polished transitions

## 🧩 Tech Stack

- React.js
- Vite
- Tailwind CSS
- Framer Motion
- React Router
- Three.js / React Three Fiber
- Lucide React

## 🏗️ Project Structure

```bash
src/
  components/      # Reusable UI and 3D experience components
  pages/           # Main application pages
  App.jsx          # Route setup and app shell
```

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm run dev
```

Then open your browser at:

```bash
http://localhost:5173
```

## 🛠️ Available Scripts

```bash
npm run dev      # start the development server
npm run build    # build the project for production
npm run preview  # preview the production build locally
```
